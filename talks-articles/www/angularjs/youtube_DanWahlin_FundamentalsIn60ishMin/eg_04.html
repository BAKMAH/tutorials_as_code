Modules

  • {{angular.name}} can be loaded from {{angular.cdnjs_uri}}
Modules can be used to create other things like Controllers, Routes,..
Modules can have 'Config' function which can be used to defined Routes.
If you have different Views which needed to be loaded into Shell page, we need to take control of what route we are on and what view is that associated with, what controller goes with that view. There comes Routes.
When you define a Route, you can define View and Controller on that.
Instead of hard-coding Controller into View. Bind it using Route.
Controller will call out to Factories (/Services/Providers/*) for CRUD.
View will have Directives, Filters.
